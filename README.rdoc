Current technologies used>

- Ruby on Rails 3.2.12
- Backbone
- Sass
- Compass
- Twitter Bootstrap
- I18n
- CoffeeScript
- jQuery
- rspec
- Cucumber
- Capybara
- Selenium
- FactoryGirl
